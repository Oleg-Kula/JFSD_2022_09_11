Приложение использует СУБД PostgreSQL
--------------------------------------------------------------------------
Настройки подключения к базе лежат в application.properties,путь к файлу:
    src/main/resources/application.properties

    Настройки по-умолчанию:
    datasource.url=jdbc:postgresql://localhost:5432/books
    username=postgres
    password=postgres
--------------------------------------------------------------------------
Перед запуском необходимо создать БД books,sql-команда:
    "create database books;"
--------------------------------------------------------------------------
Создание таблиц в БД делегировано SpringJPA, создавать их скриптом не нужно
--------------------------------------------------------------------------
Скрипт наполнения таблиц находится в папке 'sql':
    sql/insert_script.sql

!!!Выполнение скрипта наполнения необходимо для корректной работы тестов
--------------------------------------------------------------------------
Запуск приложения из IDE выполняется путем запуска StartApplication:
src/main/java/com/gmail/kulacholeg/JFSD_2022_09_11/StartApplication
